# Task Management Dashboard

## Features Implemented

- ✅ Kanban-style board (Todo, In Progress, Done)
- ✅ Drag and Drop functionality using dnd-kit
- ✅ Optimistic UI updates with rollback handling
- ✅ Task filtering (Assignee, Priority, Search)
- ✅ Debounced search input
- ✅ Task creation and editing
- ✅ Split panel view for editing tasks on click
- ✅ Modal for adding new tasks
- ✅ Keyboard support (ESC to close modal/panel)

## UX Enhancements

- Real-time task updates during drag-and-drop
- Loading state during task status updates
- Clean component structure with reusable hooks and utilities

## Tech Stack

- React
- styled-components
- dnd-kit

## Notes

Due to time constraints, some enhancements (like advanced notifications and persistence) can be added further, but core functionality is implemented and working.
